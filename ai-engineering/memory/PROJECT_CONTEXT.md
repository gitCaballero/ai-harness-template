# Project Context

## Propósito

Template para analizar y operar codebases con IA usando una constitución de evidencia, simplicidad agentic, memoria mínima, guardrails, tests y soporte de CI/CD con Harness.io.

## Alcance

- Análisis de codebases existentes.
- Diseño pragmático de soporte IA.
- Implementación controlada con tests y checklist.
- Operación con trazabilidad y memoria.

## Fuentes de verdad

- `CONSTITUTION.md`
- código y documentación observable del repositorio objetivo;
- archivos de memoria en `ai-engineering/memory/`;
- specs en `ai-engineering/specs/`;
- evaluaciones en `ai-engineering/evals/`.

## Actores base

- agente principal de análisis e implementación;
- humano aprobador para contexto funcional, producción y decisiones críticas;
- pipeline CI/CD para verificación automatizada.
